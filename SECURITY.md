# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this module, please report it by contacting:

- **Email:** security@changerplanet.com
- **Repository:** https://github.com/changerplanet/webwaka-suite-public-portal/security/advisories

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Security Standards

This module adheres to the security requirements defined in the WebWaka Ecosystem Constitution v1.4 (Ratified).

## Response Timeline

- **Critical vulnerabilities:** Response within 24 hours
- **High severity:** Response within 72 hours
- **Medium/Low severity:** Response within 7 days
